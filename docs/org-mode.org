* List[fn:plain_list] :l:
  
** Unordered List
   
  bullets: =-=, =+=, =*=
 
** Ordered List
   
  bullets: =1)=, =1.=
   
** Description
   
  start with: =::=
   
  ATTENTION: only for unordered list
   

** Exmaple
   
   1. AAA
   2. BBB
      1. aaa
      2. bbb
   3. CCC
      + eee
      + ccc
        + zzz :: explain something
        + dd :: description here
   4. DDD
   [@20] 不同的项
      * yes
      * test :: here
        * a
        * b
   
   a. AAA
   b. BBB
      a. aaa
   [@d] DDD
   
   1) AAA
   2) BBB
      1) aaa
   3) CCC
